# Weather Information App

> A Python-based application that fetches real-time weather data using an external API and displays structured output.

## Features
- Fetches live weather data for any city
- Displays temperature, weather condition, and humidity
- Emoji-based temperature representation
- Handles invalid inputs and errors gracefully

## Tech Stack
- Python
- requests library
- OpenWeatherMap API

---

## How It Works
The application sends a request to the OpenWeatherMap API using the user-provided city name. It processes the JSON response and displays relevant weather details in a readable format.
